## 34.2-1

> Consider the language $\text{GRAPH-ISOMORPHISM}$ $= \\{\langle G_1, G_2 \rangle: G_1$ and $G_2$ are isomorphic graphs$\\}$. Prove that $\text{GRAPH-ISOMORPHISM} \in \text{NP}$ by describing a polynomial-time algorithm to verify the language.

To show $\text{GRAPH-ISOMORPHISM} \in \text{NP}$, use as a certificate a bijection $f: V(G_1) \to V(G_2)$.

A verifier checks in polynomial time that:

1. $f$ is one-to-one and onto.
2. For all $u,v \in V(G_1)$, $(u,v) \in E(G_1)$ iff $(f(u),f(v)) \in E(G_2)$.

If both tests pass, accept; otherwise reject. The checks take at most $O(n^2)$ time (or polynomial in the input size), so verification is polynomial. Hence $\text{GRAPH-ISOMORPHISM} \in \text{NP}$.

## 34.2-2

> Prove that if $G$ is an undirected bipartite graph with an odd number of vertices, then $G$ is nonhamiltonian.

Let $G = (V, E)$ be a bipartite graph with bipartition $V = X \cup Y$, where $X \cap Y = \emptyset$. Assume for contradiction that:

1. The total number of vertices $n = |X| + |Y|$ is **odd**.
2. $G$ contains a **Hamiltonian cycle** $C$.

Since $G$ is bipartite, the cycle $C$ must alternate between vertices in $X$ and $Y$. Starting at a vertex in $X$, the cycle proceeds to $Y$, then back to $X$, and so on. To return to the starting vertex in $X$, the cycle must consist of an **even** number of steps. However, $n$ is odd, making it impossible to alternate perfectly and return to the starting point.

This contradiction implies that $G$ cannot have a Hamiltonian cycle when $n$ is odd.

## 34.2-3

> Show that if $\text{HAM-CYCLE} \in P$, then the problem of listing the vertices of a hamiltonian cycle, in order, is polynomial-time solvable.

We can use the following Lemma:

**Lemma:** Let $G$ be a Hamiltonian graph. If every edge in $G$ is **essential**, meaning it must appear in **every** Hamiltonian cycle of $G$, then $G$ itself must be a Hamiltonian cycle.

**Proof of Lemma (by Contradiction):** Let $C$ be a Hamiltonian cycle in $G$. Assume $G \neq C$, meaning there exists at least one edge $e \in G$ that does not belong to $C$. Thus, $e$ is **non-essential**. This contradicts the assumption that every edge in $G$ is **essential**.

**Proof:** Assume $\text{HAM-CYCLE} \in P$, meaning there exists a polynomial-time algorithm $A$ that decides whether a given graph contains a Hamiltonian cycle.

Let $G = (V, E)$ be a graph with a Hamiltonian cycle. We begin by setting $G^\ast = G$.

For each edge $e \in E$, we perform the following steps:

1. Remove $e$ from $G^\ast$, resulting in a subgraph $G'$.
2. Use algorithm $A$ to check if $G'$ still contains a Hamiltonian cycle:
    - **Case 1: $e$ is essential.** If removing $e$ from $G^\ast$ causes $G'$ to lose its Hamiltonian cycle, then $e$ must be part of every Hamiltonian cycle in $G^\ast$. In this case, $G^\ast$ remains unchanged.
    - **Case 2: $e$ is non-essential.** If $G'$ still contains a Hamiltonian cycle, then there exist Hamiltonian cycles in $G^\ast$ that do not include $e$. In this case, we safely remove $e$ from $G^\ast$, so we update $G^\ast = G'$.

The iterations ensure that $G^\ast$ remains a subgraph of $G$ that still contains a Hamiltonian cycle. Every edge that remains in the final graph must be essential for the last $G^\ast$. Thus, by the lemma above, $G^\ast$ must be a Hamiltonian cycle. Since $G^\ast$ is a subgraph of $G$, it is also a Hamiltonian cycle of $G$.

Algorithm $A$ runs in polynomial time, and since there are at most $O(n^2)$ edges in $G$, the total number of calls to $A$ is at most $O(n^2)$. Thus, the running time of the iterations is polynomial.

Finally, listing the vertices of the Hamiltonian cycle in order can be done in polynomial time. This involves starting from an arbitrary vertex in $G^\ast$ and following the Hamiltonian cycle in $G^\ast$. Regardless of whether the graph is stored with an adjacency list or adjacency matrix, this will take polynomial time.

Thus, the problem of listing the vertices of a Hamiltonian cycle in order can be solved in polynomial time.

## 34.2-4

> Prove that the class $\text{NP}$ of languages is closed under union, intersection, concatenation, and Kleene star. Discuss the closure of $\text{NP}$ under complement.

Let $L_1,L_2 \in \text{NP}$.

- **Union:** A certificate gives which language to verify and the corresponding certificate. Run the chosen verifier, so $L_1 \cup L_2 \in \text{NP}$.
- **Intersection:** A certificate is a pair $(c_1,c_2)$. Verify $x \in L_1$ with $c_1$ and $x \in L_2$ with $c_2$. Thus $L_1 \cap L_2 \in \text{NP}$.
- **Concatenation:** For $x$, certificate gives a split $x=yz$ and certificates for $y \in L_1$ and $z \in L_2$. Verify both in polynomial time, so $L_1L_2 \in \text{NP}$.
- **Kleene star:** Certificate gives a decomposition $x=x_1x_2\cdots x_m$ and certificates showing each $x_i \in L_1$. Since $m \le |x|$, total verification time is polynomial, so $L_1^* \in \text{NP}$.

For **complement**, closure is unknown in general. If $\text{NP}$ were closed under complement, then $\text{NP}=\text{co-NP}$, which is an open problem.

## 34.2-5

> Show that any language in $\text{NP}$ can be decided by an algorithm running in time $2^{O(n^k)}$ for some constant $k$.

Let $L \in \text{NP}$. Then there is a polynomial-time verifier $V(x,c)$ and a polynomial $p$ such that if $x \in L$, some certificate $c$ with $|c| \le p(|x|)$ makes $V$ accept.

A deterministic decider can try all binary strings $c$ of length at most $p(n)$, where $n=|x|$, and run $V(x,c)$ on each. It accepts if any certificate works.

There are at most $2^{p(n)}$ certificates, and each verification is polynomial. So total time is $2^{p(n)} \cdot \text{poly}(n)=2^{O(n^k)}$ for some constant $k$ (the degree of $p$).

## 34.2-6

> A **_hamiltonian path_** in a graph is a simple path that visits every vertex exactly once. Show that the language $\text{HAM-PATH}$ $= \\{\langle G, u, v \rangle:$ there is a hamiltonian path from $u$ to $v$ in graph $G\\}$ belongs to $\text{NP}$.

To prove $\text{HAM-PATH} \in \text{NP}$, use as a certificate an ordered list of vertices
$\langle v_1,v_2,\dots,v_n\rangle$.

The verifier checks in polynomial time that:

1. $v_1=u$ and $v_n=v$.
2. Every vertex of $G$ appears exactly once in the list.
3. For each $i$, $(v_i,v_{i+1}) \in E$.

If all checks pass, the list is a Hamiltonian path from $u$ to $v$; otherwise reject. These checks are polynomial, so $\text{HAM-PATH} \in \text{NP}$.

## 34.2-7

> Show that the hamiltonian-path problem from Exercise 34.2-6 can be solved in polynomial time on directed acyclic graphs. Give an efficient algorithm for the problem.

For a DAG $G=(V,E)$, solve $\text{HAM-PATH}$ from $u$ to $v$ by computing the longest path in linear time.

1. Compute a topological order of $G$.
2. Dynamic programming on that order: let $\text{dp}[x]$ be the maximum number of vertices on a path from $u$ to $x$. Initialize $\text{dp}[u]=1$, others $-\infty$.
3. For each edge $(x,y)$ in topological order, relax  
   $\text{dp}[y]=\max(\text{dp}[y], \text{dp}[x]+1)$.

Because $G$ is acyclic, this is correct and runs in $O(|V|+|E|)$.  
There is a Hamiltonian path from $u$ to $v$ iff $\text{dp}[v]=|V|$.

## 34.2-8

> Let $\phi$ be a boolean formula constructed from the boolean input variables $x_1, x_2, \dots, x_k$, negations ($\neg$), ANDs ($\wedge$), ORs ($\vee$), and parentheses. The formula $\phi$ is a **_tautology_** if it evaluates to $1$ for every assignment of $1$ and $0$ to the input variables. Define $\text{TAUTOLOGY}$ as the language of boolean formulas that are tautologies. Show that $\text{TAUTOLOGY} \in \text{co-NP}$.

Let $\overline{\text{TAUTOLOGY}}$ be the set of formulas that are **not** tautologies.  
A certificate for $\phi \in \overline{\text{TAUTOLOGY}}$ is an assignment to its variables that makes $\phi=0$.

Given $\phi$ and such an assignment, we can evaluate $\phi$ in polynomial time and verify the claim. Hence $\overline{\text{TAUTOLOGY}} \in \text{NP}$.

Therefore, by definition of co-NP, $\text{TAUTOLOGY} \in \text{co-NP}$.

## 34.2-9

> Prove that $\text P \subseteq \text{co-NP}$.

Take any language $L \in \text P$. Since $\text P$ is closed under complement, $\overline L \in \text P$.
Also, $\text P \subseteq \text{NP}$, so $\overline L \in \text{NP}$.

By definition of co-NP, this implies $L \in \text{co-NP}$. Hence $\text P \subseteq \text{co-NP}$.

## 34.2-10

> Prove that if $\text{NP} \ne \text{co-NP}$, then $\text P \ne \text{NP}$.

Prove the contrapositive: if $\text P=\text{NP}$, then $\text{NP}=\text{co-NP}$.

Indeed, if $\text P=\text{NP}$, then
$$
\text{co-NP}=\text{co-P}=\text P=\text{NP},
$$
because $\text P$ is closed under complement.

Therefore, if $\text{NP} \ne \text{co-NP}$, it must be that $\text P \ne \text{NP}$.

## 34.2-11

> Let $G$ be a connected, undirected graph with at least $3$ vertices, and let $G^3$ be the graph obtained by connecting all pairs of vertices that are connected by a path in $G$ of length at most $3$. Prove that $G^3$ is hamiltonian. ($\textit{Hint:}$ Construct a spanning tree for $G$, and use an inductive argument.)

Let $T$ be any spanning tree of $G$. Since $T$ is a subgraph of $G$, we have $T^3 \subseteq G^3$.  
So it is enough to prove $T^3$ is hamiltonian.

We prove by induction on $|V(T)|$ a stronger claim: for any rooted tree $T$ with root $r$, there is an ordering
$r=v_1,v_2,\dots,v_n$ of all vertices such that:

1. $d_T(v_i,v_{i+1}) \le 3$ for all $i$,
2. $v_n$ is a child of $r$ (or $v_n=r$ when $n=1$).

Base case $n=1$ is trivial.

For the inductive step, let the children of $r$ be $c_1,\dots,c_k$, with rooted subtrees $T_1,\dots,T_k$.  
By induction, each $T_i$ has an order starting at $c_i$ with consecutive distance at most $3$. Reverse each such order, and concatenate:
$$
r,\ \text{rev}(T_1),\ \text{rev}(T_2),\ \dots,\ \text{rev}(T_k).
$$
Inside each block, consecutive vertices stay within distance $3$. Between blocks, the jump is from $c_i$ to the first vertex of $\text{rev}(T_{i+1})$, which is within distance at most $3$ through $r$. Also, the first jump from $r$ and the final vertex $c_k$ are within distance at most $3$ from $r$.

Hence in $T^3$, every consecutive pair in this order is adjacent, and the last vertex is adjacent to the first, giving a Hamiltonian cycle in $T^3$. Therefore $G^3$ is hamiltonian.
