## 35.3-1

> Consider each of the following words as a set of letters: $\\{\text{arid}$, $\text{dash}$, $\text{drain}$, $\text{heard}$, $\text{lost}$, $\text{nose}$, $\text{shun}$, $\text{slate}$, $\text{snare}$, $\text{thread}\\}$. Show which set cover $\text{GREEDY-SET-COVER}$ produces when we break ties in favor of the word that appears ﬁrst in the dictionary.



Universe of letters:
$$
U=\{a,d,e,h,i,l,n,o,r,s,t,u\}.
$$

Set sizes (distinct letters) show the first pick is `thread` (size $6$).

After choosing `thread`, uncovered letters are $\{i,l,n,o,s,u\}$. The best gain is $3$, tied by `lost`, `nose`, `shun`; dictionary order picks `lost`.

Now uncovered letters are $\{i,n,u\}$. Best gain is $2$, tied by `drain` and `shun`; dictionary order picks `drain`.

Now only $u$ is uncovered, so pick `shun`.

Hence GREEDY-SET-COVER returns
$$
\{\texttt{thread},\ \texttt{lost},\ \texttt{drain},\ \texttt{shun}\}.
$$

## 35.3-2

> Show that the decision version of the set-covering problem is $\text{NP-complete}$ by reducing it from the vertex-cover problem.



Reduce VERTEX-COVER to SET-COVER.

Given $G=(V,E)$ and integer $k$, define:
- universe $X=E$ (one element per edge),
- for each vertex $v\in V$, a set
  $$
  S_v=\{e\in E:\text{$e$ is incident to }v\}.
  $$

Then choosing at most $k$ sets whose union is $X$ is equivalent to choosing at most $k$ vertices covering all edges.

So $(G,k)$ is a yes-instance of VERTEX-COVER iff $(X,\{S_v\}_{v\in V},k)$ is a yes-instance of SET-COVER. The transformation is polynomial, and verification is polynomial, so decision SET-COVER is NP-complete.

## 35.3-3

> Show how to implement $\text{GREEDY-SET-COVER}$ in such a way that it runs in time $O\Big(\sum_{S \in \mathcal F} |S|\Big)$.



Store for each element $x\in X$ its incidence list $\{S\in\mathcal F:x\in S\}$.
For each set $S$, keep a counter $u(S)$ = number of currently uncovered elements in $S$.
Bucket sets by counter value $0,1,\dots,\max|S|$ and keep a pointer to the highest nonempty bucket.

At each greedy step, extract a set $S$ from the highest bucket, add it to the cover, and for each newly covered element $x$, scan its incidence list and decrement $u(T)$ for each affected set $T$, moving $T$ one bucket down.

Each incidence pair $(x,S)$ is processed at most once (when $x$ becomes covered), so total work is
$$
O\!\left(\sum_{S\in\mathcal F}|S|\right).
$$

## 35.3-4

> Show that the following weaker form of Theorem 35.4 is trivially true:
>
> $$|\mathcal C| \le |\mathcal C^\*| \max\\{|S|: S \in \mathcal F\\}.$$



Let
$$
m=\max\{|S|:S\in\mathcal F\}.
$$
Any cover by $|\mathcal C^\*|$ sets covers at most $|\mathcal C^\*|\,m$ elements, so
$$
|X|\le |\mathcal C^\*|\,m.
$$

Also, any run of GREEDY-SET-COVER picks at most $|X|$ sets (each chosen set covers at least one previously uncovered element). Hence
$$
|\mathcal C|\le |X|\le |\mathcal C^\*|\,m
=|\mathcal C^\*|\max\{|S|:S\in\mathcal F\}.
$$

## 35.3-5

> $\text{GREEDY-SET-COVER}$ can return a number of different solutions, depending on how we break ties in line 4. Give a procedure $\text{BAD-SET-COVER-INSTANCE}(n)$ that returns an $n$-element instance of the set-covering problem for which, depending on how we break ties in line 4, $\text{GREEDY-SET-COVER}$ can return a number of different solutions that is exponential in $n$.



Let $n$ be given and $k=\lfloor n/3\rfloor$. Create $k$ disjoint gadgets, each with elements
$$
\{a_i,b_i,c_i\}\quad (1\le i\le k),
$$
and sets
$$
\{a_i,b_i\},\ \{a_i,c_i\},\ \{b_i,c_i\}.
$$
(If $3k<n$, add dummy elements each with a unique singleton set.)

In every gadget, all three sets initially have equal gain $2$, so tie-breaking can pick any one. Since each chosen set has exactly two of the three gadget elements, exactly one element remains uncovered in that gadget after the first pick. Either of the two remaining sets contains that last element (equal gain $1$), giving another binary tie.

Thus each gadget yields multiple independent tie choices; total number of different greedy outputs is at least $2^k=2^{\Omega(n)}$ (in fact larger). Therefore the number of possible GREEDY-SET-COVER solutions can be exponential in $n$.
